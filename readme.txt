Plants vs Smart Zombies
By: Kevin Lai

Requirements:
Python 3.6+ (For cmu_112_graphics compatibility)

How to run the game:
    Step 1: Install Modules
        Run the following three commands in your terminal if your do not have these:  
        pip install requests  
        pip install pillow  
        pip install image  
    Step 2: Starting the game
        Run the __init__ python file.
